Machine Learning and Statistical Learning

  • An introduction to statistical learning: A beginner-friendly book providing an introduction to topics related to statistical learning. With exercises in R.
  • The elements of statistical learning: A book covering topics related to statistical learning in a more advanced way.
  • Statistical Rethinking: A book introducing bayesian analysis. With exercises in R.
  • A visual introduction to hierarchical modeling: A visual explanation of hierarchical modeling.

Math

  • Mathematics for machine learning: A book introducing math useful for conducting machine learning analysis.
Linear Algebra
  • Essence of linear algebra: A set of videos providing a friendly and visual introduction to linear algebra.
  • Linear Algebra: An online course providing and in depth introduction to linear algebra. With exercises.
Calculus
  • Essence of calculus: A set of videos providing a friendly and visual introduction to calculus topics.
  • Calculus made easy: A book giving simple explanations of calculus concepts. With exercises.
Probability
  • Introduction to probability: An online course illustrating the fundamentals of probability theory. With exercises.

Research Software Development

  • Merely useful: An online book introducing research software engineering practices.
  • Python package best practices: An online lesson introducing best practices for setting up a Python package.
  • Clean Code: A book on best practices for software development.

Computer Science

  • CS50: An introduction to computer science: An online course introducting computer science and programming basic concepts. With exercises.
  • The missing semester of your CS education: An online course introducing tools not commonly taught at traditional computer science programs. With exercises.
